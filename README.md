# joke-book-cover-generate
Generates a cover in the style of the "Cambridge studies in advanced mathematics" series of books, for joke use

Cambridge University Press owns the copyright of the original cover. To avoid copyright issues, don't use it for commercial purposes!

## Usage:
```shell
python joke-book-cover-generate <Your-Cover-Title> <Author> [Font]
```
Default font is `SourceHanSansCN-Heavy.otf`, if you don't have it, will throw an error.
